# Audiobook Player

A browser-based audiobook player built with HTML, CSS and JavaScript.

## Screenshots

![Player](images/screenshot-1.png)
![Player Playing](images/screenshot-2.png)

## Features

- Play / pause, next, previous track controls
- Seek and volume sliders
- Random playback mode
- Track progress display
- Animated album art and sound wave indicator

Open `index.html` in a browser — no build step required.
